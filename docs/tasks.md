## PicTureen

Программа предназначена для работы с библиотекой изображений.

Предполагаемый сценарий использования: после установки пользователь выбирает 
папку, которая будет считаться корневой для библиотеки изображений, и использует данную программу
для просмотра изображений и управления тэгами, связанными с изображениями.

Основные возможности:
* Навешивание, удаление тэгов
* Проставление оценок
* Добавление комментария
* Поиск по тэгам, имени, размеру, оценке, комментарию (возможно, с дополнительными параметрами)
* Отображение изображений
* При просмотре можно масштабировать изображение и поворачивать его на 90 градусов
* Поиск дубликатов
* "Показать в проводнике"

Планируется поддержка форматов jpg, png и bmp.

Данная программа разрабатывается для ОС Windows и использует .NET 4.5
